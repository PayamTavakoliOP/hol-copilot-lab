# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Prettier configuration for consistent code formatting
- ESLint configuration with accessibility rules (jsx-a11y)
- EditorConfig for editor consistency
- Environment variable template (.env.example)
- Node version specification (.nvmrc)
- MIT License
- Contributing guidelines (CONTRIBUTING.md)
- GitHub Actions CI/CD workflow
- Product service layer (services/api.ts)
- Custom useProducts hook for data fetching
- ErrorBoundary component for error handling
- Comprehensive test coverage (91.51%)

### Changed

- Improved CSS with modern design system
- Enhanced accessibility compliance (WCAG 2.1 Level AA)
- Updated documentation

### Fixed

- CSS pseudo-element blocking button clicks (pointer-events issue)

## [1.0.0] - 2026-02-05

### Added

- Initial release of The Daily Harvest eCommerce platform
- Product catalog with dynamic loading
- Shopping cart with React Context
- Admin panel for sales management
- Responsive design
- Full keyboard navigation
- Screen reader support
